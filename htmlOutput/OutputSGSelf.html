2.2 吊放環片過程彎矩計算
環片內半徑 R = 2.8m
環片厚度 TH = 0.25m
環片角度 = 72°
相鄰螺栓孔角度 = 36°
環片單位重 γ = 24kN/m³
環片投影周長 L
  L = 2 * (2.8 + 0.25/2) * sin36° = 3.44 m
單位長度環片自重 w
  w = π * (3.05² - 2.8²) * 72°/360° * 24/3.44 = 6.41 kN/m
  L2 = 2 * (2.8 + 0.25/2) * sin18° = 1.81 m
  L1 = (L-L2)/2 = 0.82 m
彎矩計算
  0 ≤ X ≤ L1：M = (-wX²)/2
  Mmax1 = -2.13 kN-m (X = L1)
  L1 ≤ X ≤ L/2：M = (-wX²)/2 + X * w (X - L1)
  Mmax2 = 0.49 kN-m (X = L/2)
Mmax = max(|Mmax1|, |Mmax2|)
  = 2.13 kN-m
Vmax = (w * L/2) - ( w * L1)
  = 5.8 kN
2.3 環片現場堆置彎矩計算
環片內半徑 R = 2.8m
環片厚度 TH = 0.25m
A片環片角度 = 72°
B片環片角度 = 64.5°
K片環片角度 = 15°
環片單位重 γ = 24kN/m³
B1 環片檢核
B1環片投影周長 L
  L = 2 * (2.8 + 0.25/2) * sin32.25° = 3.12 m
B1單位長度環片自重 w
  w(B1 + K) = π * (3.05² - 2.8²) * (64.5° + 15°)/360° * 24/3.12 = 7.8 kN/m
彎矩計算
  Mmax1 = wL/2 * L/2 = 19 kN-m
A3 環片檢核
 L = 2 * (2.8 + 0.25/2) * sin36° = 3.44 m
單位徑度環片自重
  w = π * (3.05² - 2.8²) * (1)/360 * 24 = 0.31 kN/m
集中荷重 P
  P = 0.31 * (15 + 2 * 64.5 + 2 * 72)/2 = 44.11 kN
A 單位長度環片自重 Wu
  Wu = π * (3.05² - 2.8²) * 72°/360° * 24/3.44 = 6.41 kN/m
支撐反力 R
  R = 44.11 + 6.41 * 3.44/2 = 55.13 kN
彎矩計算
L1 = 0.15 m   L2 = 0.94 m   L3 = 0.63 m
1. 端點至支撐點
  0 ≤ X ≤ L1 + L2：
  Mmax2 = (-wX²)/2 - P * (X - L1)
    = - 6.41/2 * (0.15 + 0.94)² - 44.11 * 0.94
   = -45.27 kN-m   (X = L1 + L2)
2. 端點至支撐點
  (L1 + L2) ≤ X ≤ L1 + L2 + L3：
  Mmax3 = (-wX²)/2 - P * (X - L1) + R * (X - L1 -L2)
    = - 6.41/2 * (0.15 + 0.94 + 0.63)² - 44.11 * (0.94 + 0.63) + 55.13 * 0.63
    = -44 kN-m   (X = L1 + L2 + L3)
  Mmax = max(|Mmax1|, |Mmax2|, |Mmax3|)
      = 45.27 kN-m
  Vmax = 55.13 - (6.41 * 0.63)
      = 51.09 kN
2.4 環片吊裝彎矩計算
環片內半徑 R = 2.8m
環片厚度 TH = 0.25m
環片角度 = 72°
相鄰灌漿孔角度 = 24°
環片單位重 γ = 24kN/m³
環片投影周長 L
  L = 2 * (2.8 + 0.25/2) * sin36° = 3.44 m
單位長度環片自重 w
  w = π * (3.05² - 2.8²) * 72°/360° * 24/3.44 = 6.41 kN/m
  L2 = 2 * (2.8 + 0.25/2) * sin12° = 1.22 m
  L1 = (L - L2)/2 = 1.11 m
彎矩計算
0 ≤ X ≤ L1：
  Mmax1 = (-wX²)/2
    = -3.96 kN-m   (X = L1 + L2)
L1 ≤ X ≤ L/2：
  Mmax2 = (-wX²)/2 + X * w (X - L1)
    = -2.77 kN-m   (X = L/2)
Mmax = max(|Mmax1|, |Mmax2|)
    = 3.96 kN-m
VMax = (w * L/2) * (w * L1)
    = 7.13
